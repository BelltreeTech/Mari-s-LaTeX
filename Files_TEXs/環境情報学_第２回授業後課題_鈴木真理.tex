\section{私のおすすめスポット３選}
\label{sec:multicol}

\begin{itemize}
    \item \textbf{小さな公園}：\\私はSFC周辺の団地の中にある小さな公園を見つけた．この公園は、緑豊かな木々と花々に囲まれており、ベンチも設置されているため、勉強や読書に最適な場所であった．特に春には桜が満開になり、非常に美しい景色が広がっているのではないだろうか．
    \item \textbf{慶應看護学部の校舎}：\\私はSFCの外にある慶應看護学部の校舎を訪れた．この校舎は、モダンな建築デザインと広々とした空間が特徴であり、自然光がたっぷりと入るため、非常に快適な環境であった．特に、屋上からの眺めは素晴らしく、周囲の緑や空を一望できるスポットであった．
    \item \textbf{慶應藤沢イノベーションビレッジ}：\\私はSFCの近くにある慶應藤沢イノベーションビレッジを訪れた．この施設は、最新の技術とデザインが融合した空間であり、研究者が集まる場所であった．特に、屋外スペースには緑が多く、リラックスできる環境が整っており、学業の合間にリフレッシュするのに最適なスポットであった．
\end{itemize}
