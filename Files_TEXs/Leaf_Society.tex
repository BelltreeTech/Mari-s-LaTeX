\section{数理的基盤 (Mathematical Framework)}
ニューラルネットワークにおけるAffineレイヤの演算は、行列（Matrix）の積によって表現される。

\subsection{順伝播と逆伝播の定式化}
入力 $\bm{X}$、重み $\bm{W}$、バイアス $\bm{b}$ としたとき、順伝播（Forward Propagation）の出力 $\bm{Y}$ は式(\ref{eq:forward})のように定義される。

\begin{equation}
    \bm{Y} = \bm{X}\bm{W} + \bm{b}
    \label{eq:forward}
\end{equation}

また、損失関数 $L$ に対する勾配（Gradient）を計算する際、連鎖律（Chain Rule）を適用したBackward Propagationは、行列の転置（Transpose）を用いて以下のように展開される。

\begin{align}
    \frac{\partial L}{\partial \bm{X}} & = \frac{\partial L}{\partial \bm{Y}} \bm{W}^T \label{eq:back_x} \\
    \frac{\partial L}{\partial \bm{W}} & = \bm{X}^T \frac{\partial L}{\partial \bm{Y}} \label{eq:back_w}
\end{align}

式(\ref{eq:back_w})で求めた勾配を用いて、パラメータ $\bm{W}$ を更新する。

\section{実装と検証 (Implementation)}
\texttt{NumPy} のみを用いて実装したAffineレイヤのソースコードを以下に示す。

\begin{lstlisting}[language=Python, caption=Python/NumPyによるAffine Layerの完全実装, label=code:affine]
import numpy as np

class Affine:
    def __init__(self, W, b):
        self.W = W
        self.b = b
        self.x = None
        self.dW = None
        self.db = None

    def forward(self, x):
        self.x = x
        # Forward Propagation: Y = XW + b
        out = np.dot(x, self.W) + self.b 
        return out

    def backward(self, dout):
        # Backward Propagation using Transpose
        dx = np.dot(dout, self.W.T)
        self.dW = np.dot(self.x.T, dout)
        self.db = np.sum(dout, axis=0) # Handle broadcasting
        return dx
\end{lstlisting}

ソースコード\ref{code:affine}の21行目にあるように、バイアス `b` の逆伝播では次元を維持するために `axis=0` での総和（Summation）が不可欠である。
